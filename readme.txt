A chat platform for WP users to interact each other.

== Description ==
This plugin intergrate the chat feature among WordPress users with shortcode. "Create chat button" will display if shortcode is added. Chat bubble will be displayed if the plugin is activated. This plugin works only for logged-in users.

Ordered list:

1. Chat Window
2. Chat bubble
2. Light and Dark Skin

== Installation ==
1. Download and Unzip "jp-user-chat.zip" file.
1. Upload "jp-user-chat" to the "/wp-content/plugins/" directory.
1. Activate the plugin through the "Plugins" menu in WordPress.
1. Paste the shortcode [jpchat user_id='id'] to display the create chat button.

== Frequently Asked Questions ==
= Will this plugin slow down my website? =
This plugin works only for logged-in users. All the needed static files are loaded only for logged-in users. Only small two files and the default WP jQuery is using hence it is soo light weight.

= Can I clear the custom data created by this plugin =
There is a option in the plugin settings page to clear the chat custom data as well as to delete the custom tables hence it will not mess up your WordPress.
